# Marisol — Ladies Handbags E-Commerce

Modern B2C ladies handbags e-commerce store built with React, Vite, TypeScript, Tailwind CSS, and Supabase.

## Features

- Product catalog with categories, filtering, and sorting
- User authentication (login/signup)
- Shopping cart with persistent storage
- Wishlist
- Checkout with Razorpay payment integration
- GST invoice generation
- Order management
- Admin panel (products, orders, dashboard)
- WhatsApp support widget
- Responsive design

## Tech Stack

- React 18 + Vite 5 + TypeScript 5
- Tailwind CSS 3
- Supabase (Postgres, Auth, RLS)
- Razorpay (payments)
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```
